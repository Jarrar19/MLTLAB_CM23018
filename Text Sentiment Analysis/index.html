Practical 3: Text Sentiment Analysis

Course Outcome: CO2 – Analyze

Objective

To classify text as Positive or Negative using deep learning models. The practical involves training a custom Recurrent Neural Network (RNN), testing it on custom sentences, and comparing its performance with a simple Dense network.

Pre-trained Sentiment Model (Node.js Implementation)

The pre-trained sentiment model is implemented using TensorFlow.js with Node.js backend inside the pretrained folder.

Run using:
node app.js

Lab Assignments

  • Lab Assignment 1 – Train RNN Sentiment Classifier
  • Lab Assignment 2 – Test Custom Sentences & Confidence Analysis
  • Lab Assignment 3 – RNN vs Dense Network Comparison
